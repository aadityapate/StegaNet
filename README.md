# StegaNET: Secure Communication with Steganography & SAML

## Project Overview
StegaNET is a secure communication system designed to safeguard the confidentiality, integrity, and authenticity of transmitted data. It combines **Steganography** and **Cryptography** to enhance security while using **SAML** (Secure Assertion Markup Language) for user authentication and authorization. 

This project was developed as part of the curriculum in the **Department of Computer Science & Engineering** at **MIT School of Computing** under the guidance of **Prof. Pooja Oza**.

## Problem Statement
In today's digital communication landscape, protecting sensitive information is a major challenge. Traditional encryption methods are becoming vulnerable to sophisticated attacks. StegaNET aims to solve this problem by embedding encrypted messages within carrier data such as images, audio, or video files using **Steganography**. Only authorized users can access these messages, ensuring both **confidentiality** and **integrity** in secure communications.

## Key Features
- **Steganography Implementation**: Embeds encrypted messages into carrier files (images, audio, or video).
- **Cryptography**: Ensures confidentiality and integrity of the messages through encryption and decryption mechanisms.
- **User Authentication via SAML**: Restricts access to authorized users only, using **Secure Assertion Markup Language**.
- **Covert Communication**: Enhances security by making the transmission of sensitive data appear innocent and unnoticed.

## How It Works
1. **Message Encryption**: The message is encrypted using advanced cryptographic techniques.
2. **Steganographic Embedding**: The encrypted message is discreetly embedded within a carrier file (such as an image, audio, or video).
3. **User Authentication**: Users are authenticated through SAML, ensuring that only authorized users can access the hidden data.
4. **Message Extraction & Decryption**: Authorized users extract the hidden message from the carrier file and decrypt it to view the original content.
